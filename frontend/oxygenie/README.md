🌿 OxyGenie – SmartCare for Leaf & Life

OxyGenie is a modern, intelligent web application designed to provide personalized plant recommendations based on user preferences such as region, season, environment (indoor/outdoor), and health goals (oxygen-rich, medicinal, fruit-bearing, decorative).
The application combines a React-based frontend with a machine-learning powered backend to deliver accurate and meaningful plant suggestions.

🛠 Features

User Personalization – Collects user info including name, region, season, and area type.

Plant Recommendation Engine – Suggests suitable plants based on user input using ML backend.

Plant Information – Displays plant details including sunlight requirement, placement, category, oxygen output, and description.

Care Schedule & Notifications – Personalized care reminders (future enhancement).

Favorites & History – Save and track recommended plants (future enhancement).

Responsive Design – Works seamlessly across devices.

🖥 Technology Stack

Frontend: React + TypeScript + shadcn-ui + Tailwind CSS + Vite

Backend: Python FastAPI (ML-powered recommendation engine)

Database: CSV / PostgreSQL / MongoDB (for plant info and user preferences)

Machine Learning: Content-based filtering + k-NN similarity for plant recommendations
